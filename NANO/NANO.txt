
#-------------nano editor-------------#

 1) nano : Nano is a simple, beginner-friendly, terminal-based text editor in Linux, If a file doesn't exist then it will create a new one otherwise open that file.

ubuntu@ip-172-31-8-112:~/DevOps$ nano demo.txt  <-------- Here we have used this nano editor.
ubuntu@ip-172-31-8-112:~/DevOps$ ls
demo.txt  hello  hello.txt                      <------- Here we can see the "demo.txt" got created because it was not exist till that.

This is a new demo file opened by nano editor .
Herewe don't need to to press i for insert , we can directly start typing






                                                                              <---------- This is how nano editor looks, like
