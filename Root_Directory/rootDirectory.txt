
#--------Root Directory----------#
 
 1) /(Root directory) is the Linux root filesystem showing standard directories defined by the FHS(Filesystem Hierarchy Standard).
 
 2) each responsible for a specific part of system operation.
 
 3) Everything in Linux starts from /. 


ubuntu@ip-172-31-8-112:~$ cd /          <----------- This is the Root directory.
ubuntu@ip-172-31-8-112:/$ ls
bin                etc                lib64       opt   sbin                sys
bin.usr-is-merged  home               lost+found  proc  sbin.usr-is-merged  tmp
boot               lib                media       root  snap                usr
dev                lib.usr-is-merged  mnt         run   srv                 var
